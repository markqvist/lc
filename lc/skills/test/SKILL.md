# Test Skill

This is a test skill for verifying that the skill loading system works correctly.

Use this skill when testing the skill system functionality.

## Available Tools

- `echo`: Echo back a message
- `add`: Add two numbers

## Examples

```

TestSkill.add(a=5, b=3)
```
